The guarantee

Not even we can read it.

Your messages are encrypted on your device and addressed to your partner’s device key, one our server never holds. What we store is ciphertext we have no way to open. It isn’t a setting, and it isn’t a promise. It’s the only code path that exists.

(that’s what “end-to-end encrypted” means)

  • No ads. We don’t have anything to sell, including you.
  • No trackers. No analytics SDK in the app or the server. We checked the code.
  • No AI. Nothing reads your conversation to “improve” anything.

And we don’t track who you talk to, or when. There’s no one to sell it to anyway.

Don’t trust us

crypto/src/ecdh.rs
// the room key is derived on-device. the server
// never imports this module, so it can't.
pub fn derive_room_key(
    ours: &StaticSecret,
    theirs: &PublicKey,
    room_id: &str,
) -> RoomKey {
    let shared = ours.diffie_hellman(theirs);
    hkdf(shared, "littlelove.room", room_id)
}
