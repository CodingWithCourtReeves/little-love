# LittleLove App Icon

Master icon source:

- `littlelove-app-icon-master.png` — 1024x1024 PNG with transparent corners.

Regenerate platform assets after replacing the master:

```sh
./scripts/generate-app-icons.sh
```

Concept: a heart-shaped lock on a soft rounded tile, with a keyhole for end-to-end encryption. The current artwork uses the Twilight palette in `app/lib/theme/twilight.dart`, with muted rose and aubergine tones instead of the sage accent.
