LittleLove v0.3 UI MOCKS · 2026-06-10

Group rooms with familiars.

Couples stay couples — humans pair monogamously. But a chat can include any number of local AI familiars. These mocks walk the new "create chat" flow end-to-end: host picks members, host shares an invite code if a human partner is in the mix, guest reviews the roster before joining, and the three-way conversation lands with sender-tagged bubbles.
