Per-message, Telegram style: every message I send carries its own marker inside the bubble, bottom-right — a faint clock while in flight, a single heart once the server acks (sent), and a double heart once Kaitlyn reads it (read — separate PR, needs a read-receipt frame end-to-end). No collapsing for these: a run of three sent messages shows three hearts, just like Telegram's per-bubble checks. The bubble reserves a little right/bottom padding on its last line so text never collides with the mark. Failed is the one exception — it stays a collapsed text caption below the run (failed · tap to retry) so the retry affordance reads clearly instead of hiding behind a bare icon. "Sent" honestly means the server relayed it — not yet that Kaitlyn's phone received it.
