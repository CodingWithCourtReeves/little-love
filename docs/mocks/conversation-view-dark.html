Design notes — conversation view (dark)

  • Warm dark, not cold. Base #1F1814 (slight brown-black), bubbles #3E3026, ink #F0E5D3 cream. No pure neutrals anywhere. Sits in the same "evening lamp / journal" mood as the light theme rather than a developer terminal.
  • Teal lifts in dark mode. Ollie's accent shifts from forest #2E5D5A (light) to a brighter #6CB1A6 (dark) so the familiar still reads "alive" at low brightness without becoming neon.
  • Rose user bubble stays rose. Deep oxblood #4A2A23 keeps the warm coral identity in dark mode; this is the place Court is most likely to want to push back — we could flatten it to neutral if it feels too "Valentines."
  • Bot bubble keeps its left-edge teal rule. The same affordance carries through both themes, so users learn the pattern once: hex avatar + AI pill + teal edge = familiar.
  • Glow on the attachment, not the chrome. The only soft luminance in the view is the sun in the attachment photo — chrome stays matte. Keeps the "magical undertone" in the content, not the UI shell.
